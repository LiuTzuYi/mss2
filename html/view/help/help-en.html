    Items Desciption
    Video A play button show when the warning pre-defined to be captured the video during the warning occurred
    Start Time The warning starting time
    Duration (s) The warning start to end duration
    Start Speed (kph) The speed when the warning was started
    End Speed (kph) The speed when the warning was ended
    Top Speed (kph) The top speed during the warning was occurred
    Road Speed Limit (kph) The speed limit on the road section where the warning triggered
    HW (s) Headway distance to vehicle ahead (unit second)
    Nearest HW (s) The nearest headway distance to vehicle ahead (unit second)
    Driver Action The driver follow up action was taken during the warning (Turning signal/brake, etc). Check out Driver Action for more information.

You may upload a single CSV file to insert multiple items. This function is limited for vehicle detail, vehicle, vehicle group, driver and driver group maintenance. Different settings has its own format and columns. Please put the data line by line. A line should contain only one item.
  1. Vehicle
    Format: < licence plate*^ >,< vin^* >,< brand* >,< model* >,< type* >,< year* >< status* >,< department* >
    Status value only accept 'A' or 'I'.
    For vehicle type code, please check out Vehicle Type Code for more details. Default value of vehicle type code is 7 (Others).
    Example: GS123,123,Honda,Freed,80,2010,I,GS
  2. Vehicle Group
    Format: 1,< group name*^ >,< group descpt* >,< department* > or; 2,< group name* >,< licence plate* >
    All addind group rows (those start with 1) should go before add vehicle rows (those start with 2).
    Example：1,GroupA,Testing,GS
  3. Driver
    Format: < name* >,< driver code*^ >,< phone home >,< phone mobile >,< phone office >,< date of birth >,< department* >
    Please input date of birth in the format of 'YYYY-mm-dd'.
    Example: 28,11155,,,,,,GS
  4. Driver Group
    Format: 1,< group name*^ >,< group descpt* >,< department* > or; 2,< group name* >,< driver name* >
    All adding group rows (those start with 1) should go before adding driver rows (those start with 2).
    If there are drivers with same name, only the first one will be added to group.
    Example：2,GroupA,28
Click on the upload icon, select your file and click "open".

  1. Abbreviations and Symbols

Warning Type Button Marker Fullname Description
FCW Forward Collision Warning Alerts driver to react and avoid collision when the risk of an impending forward collision with vehicle/pedestrian/obstacles is too high.
LDW Lane Departure Warning Alerts driver when the car begins to swerve from the driving lane without giving correct direction indicator signal.
HMW Headway Monitoring Warning Alerts driver when the headway is less than predefined distance (in seconds) value.
AAW Abrupt Acceleration Warning Vehicle speed accelerated over predefined speed change value within a second.
ABW Abrupt Braking Warning Vehicle speed decelerated over predefined speed change value within a second.
ATW Abrupt Turning Warning The centripetal acceleration over predefined value within two seconds.
DWW Drowsing Warning Alerts driver when the driver is drowsy lasting longer than predefined duration value.
CLW Calling On Phone Warning Alerts driver when the driver is calling on phone lasting longer than predefined duration value.
DTW Distraction Warning Alerts driver when the driver is distracted and inattentive lasting longer than predefined duration value.
NDW No Driver's Portrait Warning Alerts driver when DMS visual sensor failed to detect driver's portrait lasting longer than predefined duration value.
YW Yawning Warning Alerts driver when the driver is yawning lasting longer than predefined duration value.
